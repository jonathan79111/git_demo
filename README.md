* 試著實做 demo 一樣的視覺動畫
* 素材用 ajax 取得 https://demo.onead.com.tw/cprp/xml/1121984/l.xml?pid=1121984&campid=10033&uid=1000081&guid=4262fc5d-60f4-11e9-b1f7-0242ac120004

* 點擊音量鍵需要有聲音
* view 相關邏輯寫在 src/layout.ts
* 用 src/main.ts create view 邏輯
